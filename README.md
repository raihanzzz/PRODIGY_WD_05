# 🌦️ Weather Web App

A modern and interactive **Weather Application** built using **HTML, CSS, and JavaScript** that provides real-time weather updates, forecasts, and smart UI features.

---

## 🚀 Features

- 🔍 Search weather by city name  
- 📍 Get weather using current location  
- 🌡️ Real-time temperature & weather conditions  
- 💧 Humidity, wind speed, pressure, visibility  
- ⏱️ Live date & time display  
- 📊 24-hour forecast  
- ⚠️ Error handling with retry option  
- 📱 Fully responsive design  

---

## 🧠 Key Concepts Used

- JavaScript DOM Manipulation  
- Event Handling  
- Asynchronous Programming  
- Geolocation API  
- Dynamic UI updates  
- Error handling  

---

## ✨ Advanced Features

- 🌗 Day/Night mode based on time  
- 🎤 Voice search functionality  
- 🌧️ Weather-based animations (rain/snow effects)  
- 🔄 Temperature unit toggle (°C / °F)  
- 📶 Offline support using Service Worker  
- 🎨 Smooth animations & transitions
